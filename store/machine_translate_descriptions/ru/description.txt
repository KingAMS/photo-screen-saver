Это расширение отображает экранную заставку, состоящую из слайд-шоу фотографий из нескольких источников. Поддерживаемые источники фотографий:

* Ваши альбомы Google Фото
      Примечание. Общие альбомы не поддерживаются.
* Некоторые фоновые фотографии, используемые Chromecast
* Несколько источников от 500px
* Несколько источников от reddit
* Интересные фотографии с Flickr

Примечание. Фотографии из локальной файловой системы не поддерживаются.

Для управления внешним видом слайд-шоу доступны многие параметры, в том числе:

* Подождите, пока машина перестанет работать, чтобы отобразить экранную заставку
* Интервал между фотографиями
* Режим отображения фотографий - (например, рамка, заполнение экрана)
* Анимация при переходе на фото - (например, затухание, масштабирование)
* Планировщик для управления, когда экранная заставка, дисплей и компьютер должны оставаться включенными
* Не показывать в полноэкранных окнах Chrome (например, YouTube, Netflix)
* Показать на всех дисплеях
* Время показа

Прекрасно работает на всех платформах, но может быть особенно полезен для вашего Chromebook.
