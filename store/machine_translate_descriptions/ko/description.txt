이 확장 프로그램은 여러 소스의 사진 슬라이드 쇼로 구성된 스크린 세이버를 표시합니다. 지원되는 사진 소스는 다음과 같습니다.

* Google 포토 앨범
      참고 : 공유 앨범은 지원되지 않습니다.
* Chromecast에서 사용한 일부 배경 사진
* 500px에서 여러 소스
Reddit에서 여러 출처
Flickr의 흥미로운 사진

참고 : 로컬 파일 시스템의 사진은 지원되지 않습니다.

슬라이드 쇼의 모양을 제어하는 ​​데 사용할 수있는 옵션은 다음과 같습니다.

화면 보호기를 표시하기 위해 대기 모드가 된 후 대기 시간
* 사진 시간 간격 사이
* 사진 표시 모드 (예 : 프레임, 채우기 화면)
* 사진 전환 애니메이션 - (예 : 페이드, 스케일 업)
스크린 세이버, 디스플레이 및 컴퓨터가 켜져 있어야하는시기를 제어하는 ​​스케줄러
* 전체 화면 Chrome 창 (예 : YouTube, Netflix) 위에 표시하지 않습니다.
* 모든 디스플레이에 표시
* 표시 시간

모든 플랫폼에서 훌륭하게 작동하지만 Chromebook에 특히 유용 할 수 있습니다.

가치가 있다고 생각한다면 이것을 평가하십시오. 고맙습니다.
