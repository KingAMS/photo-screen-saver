MATERIAL DESIGN
This extension has been built entirely with Polymer and vanilla JavaScript. Polymer treats everything as a Web Component I have never used things like AngularJS (or even JQuery) so I can't really evaluate the pros and cons of this approach. In addition, there is nothing in Polymer that prevents you from using it with these other solutions.
Photo Libraries

PICASA API

CHROMECAST

It is unfortunate, but the Google+ API still does not support working with photos, so the only way is with the Picasa Web API. Your Google Drive folders are visible in Google+, but I have not found an efficient way to process them so they are not available at this time. The Chromecast photos are a snapshot in time of those used for the backdrop.


Note: This program is read only for your Google photos despite what the permissions popup says. The Picasa API does not have a read only scope.
